# CLI de Gestión de Usuarios con Node.js y MySQL

## Descripción

Aplicación CLI desarrollada en Node.js para gestionar usuarios utilizando MySQL.

Permite:

- Crear usuarios
- Listar usuarios
- Actualizar usuarios
- Eliminar usuarios

---

# Tecnologías utilizadas

- Node.js
- MySQL
- mysql2
- uuid
